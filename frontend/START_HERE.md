## 🎬 Demo Scenarios

### Scenario 1: Create a Simple Node (3 minutes)

```javascript
// 1. Create file: src/nodes/greetingNode.js
import { createNode, HandleConfig } from './BaseNode';

export const GreetingNode = createNode({
  title: '👋 Greeting',
  handles: [
    HandleConfig.targetLeft('name'),
    HandleConfig.sourceRight('message')
  ],
  fields: [
    {
      name: 'greeting',
      label: 'Greeting Type',
      type: 'select',
      options: [
        { value: 'Hello', label: 'Hello' },
        { value: 'Hi', label: 'Hi' },
        { value: 'Hey', label: 'Hey' }
      ]
    }
  ]
});

// 2. Register in ui.js
import { GreetingNode } from './nodes/greetingNode';
const nodeTypes = { ..., greeting: GreetingNode };

// 3. Add to toolbar.js
<DraggableNode type='greeting' label='Greeting' icon='👋' />

// Done! Your node is ready to use.
```

### Scenario 2: Use Advanced Text Node (2 minutes)

1. Drag Text node onto canvas
2. Type: `Dear {{firstName}} {{lastName}}, welcome to {{company}}!`
3. Watch 3 handles appear automatically
4. Connect input nodes to the handles
5. Connect output to see processed text

### Scenario 3: Build a Pipeline (5 minutes)

```
[Input: name] ──→ {{name}}
                            [Text Node] ──→ [LLM] ──→ [Output]
[Input: topic] ─→ {{topic}}

Text: "Hi {{name}}, let's talk about {{topic}}."
```
